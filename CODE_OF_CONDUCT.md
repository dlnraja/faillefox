# Code de conduite de Faillefox

Faillefox est un projet open source qui se veut accueillant et inclusif.
Ce code de conduite s'applique à tous les espaces du projet : issues, PR,
discussions, commits, documentation, et toute interaction communautaire.

## Notre engagement

Nous nous engageons à offrir un environnement accueillant, sûr et
respectueux pour tous, sans distinction de niveau technique, d'identité,
d'orientation, d'apparence, de nationalité, de religion, ou de tout autre
critère.

## Nos standards

**Comportements attendus** :
- Faire preuve d'empathie, de respect et de bienveillance.
- Accepter les retours constructifs avec humilité.
- Se concentrer sur ce qui est le mieux pour la communauté.
- Admettre ses erreurs et s'excuser quand on s'est trompé.

**Comportements inacceptables** :
- Harcèlement, insultes, commentaires désobligeants (personnels ou techniques).
- Trolling, provocation, ou messages purement destructeurs.
- Divulgation d'informations privées sans consentement.
- Tout comportement qui rendrait l'espace hostile.

## Champ d'application

Ce code s'applique dans tous les espaces du projet (dépôt GitHub, issues,
PR, discussions), ainsi que dans tout espace public où un individu représente
le projet.

## Application

Les violations peuvent être signalées en privé via
[GitHub Security Advisories](https://github.com/dlnraja/faillefox/security/advisories/new)
ou en ouvrant une issue confidentielle. Les mainteneurs évalueront chaque
signalement et prendront les mesures appropriées (avertissement, blocage,
exclusion).

## Sources
